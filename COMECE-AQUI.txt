1️⃣ INSTALAR DEPENDÊNCIAS (5 minutos)
   
   Abra o PowerShell nesta pasta e execute:
   
   cd C:\Users\kaiox\autopecas-marketplace
   npm install

   Aguarde a instalação...

───────────────────────────────────────────────────────────────

2️⃣ CONFIGURAR FIREBASE (10 minutos)

   A. Acesse: https://console.firebase.google.com/
   
   B. Crie um novo projeto:
      - Nome: "autopecas-maringa"
      - Desabilite Analytics (opcional)
   
   C. Ative os serviços:
      
      Authentication:
      ✓ Menu → Authentication → Começar
      ✓ Ative "E-mail/senha"
      
      Firestore Database:
      ✓ Menu → Firestore Database → Criar banco de dados
      ✓ Modo: "Iniciar em modo de teste"
      ✓ Localização: "southamerica-east1 (São Paulo)"
      
      Storage:
      ✓ Menu → Storage → Começar
      ✓ Modo: "Iniciar em modo de teste"
   
   D. Obtenha as credenciais:
      ✓ Clique em ⚙️ → Configurações do projeto
      ✓ Role até "Seus aplicativos"
      ✓ Clique no ícone Web (</>)
      ✓ Registre o app (nome: "webapp")
      ✓ COPIE as credenciais que aparecem

───────────────────────────────────────────────────────────────

3️⃣ CONFIGURAR VARIÁVEIS DE AMBIENTE (2 minutos)

   A. Copie o arquivo de exemplo:
      
      copy .env.local.example .env.local
   
   B. Abra o arquivo .env.local com Bloco de Notas
   
   C. Cole as credenciais do Firebase:
      
      NEXT_PUBLIC_FIREBASE_API_KEY=sua_api_key_aqui
      NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=seu_projeto.firebaseapp.com
      NEXT_PUBLIC_FIREBASE_PROJECT_ID=seu_projeto_id
      NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=seu_projeto.appspot.com
      NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=123456789
      NEXT_PUBLIC_FIREBASE_APP_ID=1:123456789:web:abc123
   
   D. Salve e feche

───────────────────────────────────────────────────────────────

4️⃣ INICIAR O SERVIDOR (1 minuto)

   No PowerShell, execute:
   
   npm run dev
   
   Aguarde aparecer: "Ready in X seconds"
   
   Acesse: http://localhost:3000

───────────────────────────────────────────────────────────────

5️⃣ TESTAR O SISTEMA

═══════════════════════════════════════════════════════════════
  PRECISA DE AJUDA?
═══════════════════════════════════════════════════════════════

❌ Erro "Firebase not initialized"
   → Verifique o arquivo .env.local

❌ Pedidos não aparecem
   → Verifique regras de segurança do Firestore

❌ npm install falhou
   → Execute: npm cache clean --force
   → Depois: npm install

❌ Porta 3000 em uso
   → Mude a porta: npm run dev -- -p 3001
